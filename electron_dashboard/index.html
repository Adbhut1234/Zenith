API Keys & Setup

Complete guide to acquiring the necessary API keys for Zenith

1. Google Gemini API (AI Brain)
  • Go to Google AI Studio and sign in.
  • Click Create API key.
  • Copy the key and paste it into the Google Gemini API Key field in the Configurations tab.
2. LiveKit Cloud (Voice & Audio)
  • Go to LiveKit Cloud and sign up.
  • Create a new project.
  • Navigate to Settings > Keys in your project dashboard.
  • Click Add New Key. You will receive a WebSocket URL, an API Key, and an API Secret.
  • Copy these three values into the corresponding LiveKit fields in the Configurations tab.
3. Gmail App Passwords (Optional)
  • Go to your Google Account Security.
  • Ensure 2-Step Verification is enabled.
  • Search for App Passwords and create a new one (name it "Zenith").
  • Copy the 16-character password into the Google App Password field.
4. Mem0 Platform (Long-Term Memory)
  • Go to Mem0 Platform and sign in.
  • Generate a new API key from the dashboard.
  • Copy and paste it into the Mem0 API Key field.
